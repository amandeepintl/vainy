# Vainy
Vainy is a sleek, modern, and beautiful hardware performance dashboard and power management interface designed for Lenovo LOQ and Legion laptops running Linux.

## Core Features
- **Real-Time Telemetry**: Precision monitoring for CPU core temps, GPU usage, SSD temperature, and Motherboard diagnostics.
- **Custom Fan Curves**: Visual interface supporting linear interpolation to dynamically control fan speeds up to 5800 RPM.
- **Power profile Management**: Integrated with Linux ACPI platform profiles to auto-switch between performance and low-power modes.
- **Backlight Keyboard Controls**: RGB brightness, color mode selector, and timeout controls.
